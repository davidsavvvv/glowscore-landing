Sidebar nav row with optional Lucide icon.

```jsx
<NavItem icon={<Home size={16} />} href="/">Accueil</NavItem>
<NavItem icon={<LayoutDashboard size={16} />} active>Mes analyses</NavItem>
<NavItem icon={<Settings size={16} />}>Paramètres</NavItem>
```

Active rows get a faint white fill; idle rows brighten on hover.
